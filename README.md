# 博報堂 パーソナルコアシート Webアプリ

## 📋 概要

博報堂の採用活動で使用される「パーソナルコアシート」をWebブラウザ上で再現したアプリケーションです。  
A4サイズのシート上で直接入力・編集ができ、PDFとして出力（印刷）できます。

## ✨ 主な機能

### 1. **WYSIWYG編集（見たまま編集）**
- 実際のシートと同じレイアウト上で直接テキストを入力・編集できます
- プレビュー画面と入力フォームが一体化しています

### 2. **入力項目**
- **【My Vision】**: 中長期的なビジョンを最大3つ、各30字以内で入力
- **【My Engine】**: 
  - スローガン（30字以内）
  - 原動力（最大3つ、各30字以内）
- **【My Episode】**: 具体的なエピソードを最大6つ、各80字以内で入力
  - 【from】欄（8字以内）も入力可能

### 3. **文字数バリデーション**
- 文字数制限を超えると、入力欄の外枠が**赤色**で表示されます
- PDF出力時に文字数オーバーがあると、「文字数オーバーです」とアラートが表示され、PDFは出力されません

### 4. **自動保存機能**
- 入力内容は自動的にブラウザのローカルストレージに保存されます
- ページをリロードしても、最後に入力した内容が自動的に復元されます

### 5. **PDF出力**
- 「PDF出力 / 印刷」ボタンをクリックすると、ブラウザの印刷機能が起動します
- 印刷時は、入力用のUI要素（ボタン、プレースホルダーなど）が自動的に非表示になります
- A4サイズ（210mm × 297mm）に最適化されています

### 6. **スマートフォン対応**
- スマートフォンでもPCと同じサイズで表示されます
- レスポンシブデザインではなく、固定サイズで表示されるため、横スクロールで全体を確認できます

## 🚀 セットアップ方法

### 必要な環境
- Node.js 18以上
- npm または yarn

### インストール手順

1. **依存パッケージのインストール**
   ```bash
   npm install
   ```

2. **開発サーバーの起動**
   ```bash
   npm run dev
   ```

3. **ブラウザでアクセス**
   - 通常は `http://localhost:5173` でアクセスできます
   - 表示されたURLをブラウザで開いてください

### ビルド（本番用）

```bash
npm run build
```

ビルドされたファイルは `dist` フォルダに生成されます。

## 📝 使い方

### 基本的な使い方

1. **入力**
   - 各セクションの入力欄にテキストを入力します
   - 入力内容は自動的に保存されます

2. **文字数確認**
   - 文字数制限を超えると、入力欄の外枠が赤色になります
   - 文字数制限内に収める必要があります

3. **PDF出力**
   - 右上の「PDF出力 / 印刷」ボタンをクリックします
   - ブラウザの印刷ダイアログが表示されます
   - 「PDFに保存」を選択して保存できます

### 文字数制限

| 項目 | 制限 |
|------|------|
| My Vision | 各30字以内 |
| My Engine スローガン | 30字以内 |
| My Engine 原動力 | 各30字以内 |
| My Episode エピソード | 各80字以内 |
| My Episode 【from】 | 8字以内 |

⚠️ **注意**: 文字数制限を超えていると、PDF出力時にエラーが表示され、PDFは出力されません。

## 🎨 デザイン特徴

- **A4サイズ**: 210mm × 297mmのA4用紙サイズで設計
- **フォント**: 見出しはゴシック体、本文もゴシック体で統一
- **装飾**: 
  - My Visionセクションには斜めのグレー背景
  - My Engineセクションには中央に薄い円形の装飾
  - My Episodeセクションには上部が尖った形状

## 🔧 トラブルシューティング

### 入力内容が保存されない
- ブラウザのローカルストレージが有効になっているか確認してください
- プライベートモードでは保存されません

### PDF出力ができない
- 文字数制限を超えている入力欄がないか確認してください
- ブラウザの印刷機能がブロックされていないか確認してください

### スマートフォンで表示が崩れる
- 横スクロールで全体を確認できます
- 画面を横向きにすると見やすくなります

### データ保存形式

入力データは以下の形式でローカルストレージに保存されます：

```json
{
  "visions": ["", "", ""],
  "engineSlogan": "",
  "engines": ["", "", ""],
  "episodes": [
    { "from": "", "text": "" },
    ...
  ]
}
```

保存キー: `pcs-sheet-data`
